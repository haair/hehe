Tài liệu API

Giới thiệu

Đây là tài liệu cho các API quản lý học sinh, bao gồm các chức năng: lấy danh sách, thêm, sửa, xóa và tìm kiếm học sinh.

Cơ sở: https://student-api-451c.onrender.com

Lưu ý: Tất cả các API (trừ /api/generate-api-key) yêu cầu header X-API-Key. Sử dụng nút dưới đây để tạo API key.

3. Lấy thông tin học sinh theo ID

GET /api/students/:id

Mô tả: Lấy thông tin chi tiết của một học sinh theo ID.

Header:

  • X-API-Key: API key của bạn

Tham số:

  • id: ID của học sinh (số nguyên)

Response:

{
    "id": 1,
    "ho_ten": "Nguyen Van A",
    "ngay_sinh": "01/01/2000",
    "gioi_tinh": "Nam",
    "dia_chi": "Hà Nội",
    "fb_url": "https://facebook.com/nguyenvana"
}

Status Codes:

  • 200: Thành công
  • 400: ID không hợp lệ
  • 401: Thiếu hoặc sai API key
  • 403: API key không hoạt động
  • 404: Không tìm thấy học sinh
  • 500: Lỗi server

4. Thêm học sinh mới

POST /api/students

Mô tả: Thêm một học sinh mới vào cơ sở dữ liệu.

Header:

  • X-API-Key: API key của bạn

Body:

{
    "ho_ten": "Nguyen Van B",
    "ngay_sinh": "02/02/2001",
    "gioi_tinh": "Nữ",
    "dia_chi": "Hồ Chí Minh",
    "fb_url": "https://facebook.com/nguyenvanb"
}

Response:

{
    "message": "Student added successfully",
    "student": {
        "id": 2,
        "ho_ten": "Nguyen Van B",
        "ngay_sinh": "02/02/2001",
        "gioi_tinh": "Nữ",
        "dia_chi": "Hồ Chí Minh",
        "fb_url": "https://facebook.com/nguyenvanb"
    }
}

Status Codes:

  • 201: Thành công
  • 400: Thiếu trường bắt buộc
  • 401: Thiếu hoặc sai API key
  • 403: API key không hoạt động
  • 500: Lỗi server

7. Tìm kiếm học sinh

GET /api/students/search

Mô tả: Tìm kiếm học sinh theo họ tên, giới tính hoặc địa chỉ.

Header:

  • X-API-Key: API key của bạn

Query Parameters:

  • ho_ten: Họ tên (tùy chọn)
  • gioi_tinh: Giới tính (tùy chọn)
  • dia_chi: Địa chỉ (tùy chọn)

Ví dụ: /api/students/search?ho_ten=Nguyen&gioi_tinh=Nam

Response:

[
    {
        "id": 1,
        "ho_ten": "Nguyen Van A",
        "ngay_sinh": "01/01/2000",
        "gioi_tinh": "Nam",
        "dia_chi": "Hà Nội",
        "fb_url": "https://facebook.com/nguyenvana"
    },
    ...
]

Status Codes:

  • 200: Thành công
  • 401: Thiếu hoặc sai API key
  • 403: API key không hoạt động
  • 500: Lỗi server
